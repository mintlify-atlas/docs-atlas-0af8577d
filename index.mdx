  </div>

  <Steps>
    <Step title="Install dependencies">
      Clone the repository and install the required packages.
      
      <CodeGroup>
      ```bash npm
      npm install
      ```
      
      ```bash yarn
      yarn install
      ```
      
      ```bash pnpm
      pnpm install
      ```
      </CodeGroup>
    </Step>

    <Step title="Configure environment">
      Copy the example environment file and configure your settings.
      
      ```bash
      cp .env.example .env
      ```
      
      Set the required environment variables:
      
      ```bash .env
      NODE_ENV=development
      PORT=3000
      CORS_ORIGINS=http://localhost:3000,http://localhost:5173
      MAX_FILE_SIZE=10485760
      MAX_EMPLOYEES=1000
      ```
    </Step>

    <Step title="Start the server">
      Launch the development server and start generating contracts.
      
      ```bash
      npm run dev
      ```
      
      The API will be available at `http://localhost:3000/api`
    </Step>

    <Step title="Generate your first contract">
      Make a request to preview a contract PDF.
      
      ```bash
      curl -X POST http://localhost:3000/api/contracts/preview \
        -H "Content-Type: application/json" \
        -d '{
          "dni": "12345678",
          "name": "Juan",
          "lastNameFather": "Perez",
          "lastNameMother": "Garcia",
          "contractType": "PLANILLA",
          "position": "Developer",
          "salary": 3000,
          "entryDate": "2024-01-15",
          "endDate": "2025-01-15"
        }'
      ```
      
      <Note>The response will be a PDF document that you can view in your browser.</Note>
    </Step>
  </Steps>
</div>
